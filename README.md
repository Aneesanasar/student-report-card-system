Student Report Card Management System

This is a Java-based desktop application developed as an academic group project for managing student records and generating report cards.

 Features

- User-friendly GUI built using Java Swing
- Student and Teacher interface
- Add, update, and delete student records
- Automatic calculation of:
  - Total marks
  - Percentage
  - Grade
- View individual student report cards
- Print report cards (PDF via system print option)


 Technologies Used

- Java (JDK)
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
- ArrayList (for data storage)



 Project Description

This system allows teachers to enter student details such as name, roll number, class, section, and marks.
The application automatically calculates the total marks and assigns grades based on performance.

It also provides options to view formatted report cards and print them.



📂 How to Run

1. Install Java (JDK)
2. Open the project in any Java IDE (like VS Code / IntelliJ / Eclipse)
3. Compile and run:
   javac CompleteReportCardSystem.java
java CompleteReportCardSystem
